🖼️ 随机图片 API 接口文档

支持按分类筛选、多分类混合筛选的随机图片分发服务。

接口基础地址

https://t.ziworld.top/api

1. 随机获取单个分类图片

通过在 URL 中添加 category 参数指定单个分类名称。

示例：获取“原神”分类图片
https://t.ziworld.top/api?category=原神

2. 随机获取多个分类图片

通过在 URL 中使用英文逗号 , 分隔多个分类名称，系统将在这些分类中随机抽取。

示例：从“原神”、“PC”、“二次元”中随机获取
https://t.ziworld.top/api?category=原神,PC,二次元
⚠️ 注意事项：
  • 必须使用英文逗号 (,) 进行分隔，不支持中文逗号。
  • 分类名称区分大小写（取决于后台数据录入情况）。
  • 分类有：PC、背景、二次元、移动端、手机壁纸、横版壁纸、头像、萌图MP、萌图PC、原神、崩坏、鸣潮、video、未归类等，api请求分类名与主页中的分类同名

3. 进阶参数说明 (JSON 模式)

如果你需要在前端获取图片地址而不是直接显示图片，可以添加 type=json。

参数 是否必填 说明
category 否 分类名称。不填默认随机全库，多个分类用英文逗号分隔。
type 否 默认为空（直接重定向图片）。
设为 json 时返回 JSON 数据。

JSON 响应示例：

{ "success": true, "url": "https://t.ziworld.top/images/genshin/001.jpg", "count": 150 }
